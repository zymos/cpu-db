cpu-db.info




Access: 
	View the database at http://cpu-db.info or open the CSV files with your 
	favorite spreadsheet editor.

Description:
	A set of tables that describe CPUs, MCUs, and DSPs and their various
	parameters.  

Goals:
	Create a complete set of all the varients of common and collectable 
	CPU, MCUs and DSPs.

	b, B: bit, byte 
	um, nm: micrometers, nanometers
	mm^2, mm^3: square millimeters, cubic millimeters
	V, W, C: Volts, Watts, degrees Celsius
	kHz, MHz, GHz: kilo-Hertz, mega-Hertz, giga-Hertz
	KiB, MiB, GiB: kilobytes, megabytes, gigabytes
	MiB/s, GiB/s: megabytes per second, gigabytes per second
	k, M: thousand, million
	$: dollors US
	date format: 13 April 2001
